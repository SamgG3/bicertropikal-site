# Biçer Tropikal

Biçer Tropikal için hazırlanmış bağımsız, statik ve responsive tanıtım/katalog web sitesi.

## Açılış

`index.html` dosyasını tarayıcıda açmanız yeterli.

## İçerik

- Ana sayfa, hakkımızda, katalog, ürün detay, meyve kutuları, saklama rehberi, tarifler, blog, SSS, galeri, iletişim, kurumsal talep, 404 ve boş arama ekranı
- Gizlilik, KVKK, çerez ve kullanım koşulları sayfaları
- Canlı arama, katalog filtreleri, breadcrumb, scroll to top, dark mode, çerez banner
- Admin paneli production ortamında gizlidir; yalnızca yerel geliştirme ortamında kullanılacak şekilde bırakılmıştır.

Bu proje satış, üyelik, ödeme, kargo, adres veya kart bilgisi toplama akışı içermez.
